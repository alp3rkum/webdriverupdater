web Driver Updater for Selenium Projects

One Module To Make Sure Your Web Scraper Remains Updated



web Driver Updater is a compact and stylized Python module that turns updating your web drivers from a chore to a single-click job. All you have to do is to choose which Web Driver you want to download, and let web Driver Updater do the rest. You don't even have to choose where do you want to extract the archives in, the web Driver Updater does it for you.

Requirements

web Driver Updater requires the following to run on your computer:

  • A Windows operating system (sorry Linux and Mac)
  • Python3 (ideally 3.10 and later)
  • Colorama (for interface)
  • keyboard (for menu inputs)

The CMD version of the webDriverUpdater doesn't need Colorama and keyboard, as it is run according to a single argument it gets. The argument takes the values of:

  • --edge (to install Edge Driver)
  • --chrome (to install Chrome Driver)
  • --firefox(or --gecko) (to install Firefox Driver)
  • --help (in case you forget how to use it)

Usage of the CMD version: python webDriverUpdater_cmd.py --argument

Supported Browsers

Microsoft Edge
Google Chrome
Mozilla Firefox
